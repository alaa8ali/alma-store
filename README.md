# متجر Alma الإلكتروني

متجر إلكتروني متكامل ومتعدد الأقسام مع لوحة تحكم إدارية متقدمة.

## 🌟 الميزات الرئيسية

### 🏪 أقسام متعددة
- **🛒 مواد غذائية وبقالة** - مشروبات، حلويات، بسكويت، منتجات ألبان، فواكه وخضار
- **👕 ملابس وأزياء** - رجالية، نسائية، أطفال، رياضية، أحذية، إكسسوارات  
- **📱 إلكترونيات وتقنية** - هواتف، شواحن، سماعات، حاسوب، ألعاب، منزل ذكي

### 🎛️ لوحة تحكم متكاملة
- **📊 لوحة المعلومات** - إحصائيات شاملة ومرئية
- **🏗️ إدارة الأقسام** - إضافة/تعديل/حذف الأقسام الرئيسية
- **📦 إدارة المنتجات** - إضافة/تعديل/حذف المنتجات مع رفع الصور
- **🛍️ إدارة الطلبات** - متابعة وتحديث حالة الطلبات
- **👥 إدارة المستخدمين** - (للمدير فقط)
- **⚙️ الإعدادات** - تخصيص المتجر والتنبيهات

### 🔧 ميزات تقنية
- **🎨 تصميم متجاوب** - يعمل على جميع الأجهزة
- **🔍 فلترة ذكية** - حسب القسم والتصنيف
- **📈 إحصائيات مفصلة** - مبيعات، مخزون، أداء
- **🖼️ رفع الصور** - للمنتجات والتصنيفات
- **🔐 نظام أمان** - صلاحيات متعددة المستويات
- **📲 إرسال واتساب** - مع تحديد الموقع

## 🚀 التقنيات المستخدمة

- **React 18** - مكتبة واجهة المستخدم
- **TypeScript** - للكتابة الآمنة
- **Tailwind CSS** - للتصميم المتجاوب
- **Vite** - أداة البناء السريعة
- **Lucide React** - مكتبة الأيقونات
- **React Hot Toast** - للإشعارات

## 📦 التثبيت والتشغيل

### المتطلبات
- Node.js 18+ 
- npm أو yarn

### خطوات التثبيت

1. **استنساخ المشروع**
```bash
git clone https://github.com/yourusername/alma-store.git
cd alma-store
```

2. **تثبيت التبعيات**
```bash
npm install
```

3. **تشغيل المشروع**
```bash
npm run dev
```

4. **فتح المتصفح**
```
http://localhost:5173
```

## 🔐 الوصول للوحة التحكم

### طريقة الدخول الآمنة
```
http://localhost:5173?admin=true
```

### بيانات الدخول التجريبية
- **المدير**: `admin` / `alma2024`
- **الموظف**: `employee` / `emp2024`

```
src/
```

## 🔧 التخصيص

### إضافة منتجات جديدة
1. ادخل للوحة التحكم
2. اذهب إلى "إدارة المنتجات"
3. اضغط "إضافة منتج"

### إضافة أقسام جديدة
1. ادخل للوحة التحكم
2. اذهب إلى "إدارة الأقسام"
3. اضغط "إضافة قسم"

### تخصيص الألوان والتصميم
عدّل ملف `tailwind.config.js` لتخصيص الألوان والخطوط.
